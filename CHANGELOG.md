# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-11

- Created a pure C Win32 app with an oval-shaped window.
- Added a native `CLOSE` button and drag-to-move behavior.
- Added `build.bat` and local toolchain support via `w64devkit`.

## [0.2.0] - 2026-03-11

- Moved the original basic oval sample into `basic/`.
- Added `drivenbyimage/` with a bitmap-driven window region sample.
- Updated the top-level README for the multi-example layout.
